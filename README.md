## 👨‍💻 About

```ascii
┌─[inventory@github]─[~]
│
│ > cat about.txt
├─> Developer and sysadmin who builds Android apps, websites,
├─> Discord bots, APIs, and whatever else seems fun to implement.
│
├─> Started with simple automation scripts and somehow ended up
└─> running entire self-hosted infrastructures at 3am.
```

**Currently working on:**
- 🤖 Discord bots with actual useful features ([HazeBot](https://github.com/inventory69/HazeBot), [Void-Bot](https://github.com/Hyphonical/Void-Bot))
- 📱 Android apps that work offline ([Simple Notes Sync](https://github.com/inventory69/simple-notes-sync), [Haze-Notiz](https://github.com/inventory69/haze-notiz))
- 🌐 Self-hosted services and infrastructure
- 🎨 Web projects with Astro and modern frameworks

**Philosophy:**
- Self-host everything (because why pay for cloud when you can maintain servers at 3am?)
- Build tools that solve real problems
- If it can be automated, automate it
- "It works on my machine" is not an acceptable answer
